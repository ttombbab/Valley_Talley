# Valley Talley Traders

A web application for trading items between users. Users can trade items directly or use Valley Talley coins as 
currency.

## Features

- User registration and authentication
- Item listing with descriptions and categories
- Item-to-item trading
- Valley Talley digital currency system
- Trade proposal and acceptance workflow

## Technologies Used

- Python (Flask framework)
- SQLite database
- HTML/CSS with Bootstrap 5
- JavaScript

## How to Run Locally

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
4. Visit `http://localhost:5000` in your browser

## Database

The application uses SQLite which is created automatically on first run.
